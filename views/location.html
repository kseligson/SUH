ABOUT

The name of our application is SUH DUDE which is a parody of a popular meme and essentially stands for Start Up Here. The purpose of this application is to be a data center for eager entrepreneurs who want to utilize the data we provide to come up with the next big thing needed for San Diego. Find all the information you need to start the next business that will change the world.
